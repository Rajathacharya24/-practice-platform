# Java Interview Crack Platform

Simple offline interview preparation platform for Java developer roles.

## Features

- 55 most asked and probable interview questions
- Categories for technical + HR/package preparation
- Search and filter by category and difficulty
- Mark questions as Practiced and Confident
- Add your own custom questions and answers
- Coding Round Practice section with editor-style typing area
- 30-minute coding timer with start/pause and reset
- Save code drafts per coding problem
- Show/Hide sample Java solutions for coding round questions
- Local browser storage (your progress stays on your machine)

## Categories Included

- Core Java
- OOP
- Collections
- JVM
- Multithreading
- Exception Handling
- Java 8+
- Spring Basics
- SQL & DB
- System Design Basics
- Testing
- Project Discussion
- HR & Package

## How To Run

1. Open index.html in your browser.
2. Use Quick Practice to jump to random questions.
3. Use filters to prepare category-wise.
4. Open answers and explanations to self-check.
5. Mark progress as Practiced / Confident.
6. Use Coding Round Practice to type Java solutions.

## Coding Round Usage

1. Select a coding question.
2. Start 30-minute timer.
3. Write your Java solution in editor area.
4. Save Draft to keep your progress.
5. Use Show Sample Solution only after attempting yourself.

## Package/HR Preparation Tips

- Keep salary expectation as a researched range, not one fixed number.
- Show flexibility and interest in learning + impact.
- In negotiation, be polite and value-focused.
- Highlight projects, ownership, and problem-solving with examples.

## Suggested Daily Routine (45 mins)

1. 15 mins: Revise 6 theory questions.
2. 20 mins: Solve one coding question in timer mode.
3. 10 mins: Practice one HR/package answer aloud.
