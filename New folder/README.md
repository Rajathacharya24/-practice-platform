# Java Interview Crack Platform

A simple offline platform for BCA students who want to become Java developers.

## What this platform gives you

- 30 commonly asked Java interview theory questions
- 10-question MCQ mock quiz
- Search and filter by category and difficulty
- Mark each question as practiced or confident
- Add your own questions and answers
- Local browser storage (your progress stays on your machine)

## How to run

1. Open `index.html` in your browser.
2. Start with **Quick Practice**.
3. Read question, speak your answer loudly.
4. Click **Show Ideal Answer** and compare.
5. Mark **Practiced** or **I Am Confident**.
6. Take **10-Question Quiz** daily.

## Daily 30-minute interview routine

1. 10 min: Revise 5 easy + medium questions.
2. 10 min: Explain 2 hard questions in your own words.
3. 10 min: Attempt one mock quiz.

## Most asked Java areas to master

- Core Java and OOP
- Collections internals (`HashMap`, `ArrayList`, `Set`)
- Exception handling
- Multithreading basics (`synchronized`, `wait/notify`, deadlock)
- Java 8 features (lambda, stream, optional)
- JVM basics and garbage collection
- Spring basics (DI, annotations)
- Basic SQL for backend interviews

## Bonus interview tips

- Keep answers short: definition + example + when to use.
- If you do not know exact answer, explain related concept confidently.
- Practice writing small code snippets on paper/editor.
- Build one mini Java project and explain architecture in interview.
